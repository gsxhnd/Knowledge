# CMake

CMake 是一个管理源代码构建的工具。最初，CMake 被设计为 Makefile 各种方言（语言）的生成器，如今 CMake 生成 Ninja 等现代构建系统以及 Visual Studio 和 Xcode 等 IDE 的项目文件。

CMake 广泛用于 C 和 C++ 语言，但它也可用于构建其他语言的源代码。
